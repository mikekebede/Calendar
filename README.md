# Calendar

### App Description
This program shows a working calander view of the time in the gregorian calendar starting from 1753. It is aa simplified we can input our day in various formats when running 
them on C.
Some of them are:  
$ cal -m 11 2010
$ cal -m 13 2011
$ cal -m mar 2012
$ cal -m maRcHy 2013
$ cal -m 3 2014
$ cal -m MArCh
$ cal -m 5
$ cal 18
$ cal 2018
$ cal
